# 📇 Card de Perfil Profissional

O objetivo principal foi a criação de um componente de interface responsivo utilizando boas práticas de semântica e acessibilidade.

## 🛠️ Especificações Técnicas
O projeto foca em fundamentos de estruturação e estilização moderna:

* **HTML5 Semântico:** Substituição de contêineres genéricos por tags estruturais (`<main>`, `<article>`, `<section>`), melhorando o SEO e a leitura por tecnologias assistivas.
* **CSS3 & Flexbox:** Implementação de layout flexível para centralização de componentes e organização de seções internas (eixos principal e transversal).
* **Acessibilidade (Protanopia):** Design planejado para usuários com daltonismo. A interface utiliza uma paleta segura (azul, ciano e âmbar) e suporte visual duplo (ícones + texto), garantindo que a cor não seja o único meio de transmitir informação.
* **Arquitetura:** Separação total entre estrutura (HTML) e estilo (CSS), sem o uso de estilos *inline*.

## 🚀 O Projeto
O card apresenta informações profissionais, stack tecnológica e links de contato.

* **Status:** O ponto pulsante no avatar indica disponibilidade para oportunidades.
* **Stack:** Exibição de competências em Python, Flask, Docker, AWS e automação com N8N.

## 🧠 Relato de Experiência
Dentro do arquivo `index.html`, há uma seção de comentários detalhando o processo de pesquisa sobre Flexbox, as dificuldades encontradas e as soluções aplicadas para manter a acessibilidade visual do projeto.
